**AI World Tracker** is a comprehensive platform for tracking and analyzing global Artificial Intelligence trends. It automatically collects data from multiple sources such as arXiv, GitHub, tech media, and blogs. Utilizing intelligent classification algorithms, it categorizes content into various dimensions (Research, Product, Market, etc.) and generates visual trend analysis reports and web pages.

This project also features a unique **Manual Review & Learning Feedback Mechanism**, which continuously optimizes the accuracy of the classification algorithms based on user feedback.
