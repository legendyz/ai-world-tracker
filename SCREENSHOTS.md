## üì• How to Add Screenshots

To include actual screenshots in your GitHub repository:

1. **Create Screenshots Directory**:
   ```bash
   mkdir screenshots
   ```

2. **Capture Screenshots**:
   - Take high-quality screenshots of each interface
   - Use consistent browser/terminal settings
   - Ensure good contrast and readability

3. **Optimize Images**:
   - Save as PNG for crisp screenshots
   - Compress to reasonable file sizes (< 500KB each)
   - Use descriptive filenames

4. **Update README**:
   - Add screenshot references in the main README
   - Use relative paths: `![Dashboard](screenshots/dashboard-main.png)`

5. **Create Animated Demos** (Optional):
   - Use tools like ScreenToGif for workflow demos
   - Show data collection and analysis process
   - Keep GIFs under 5MB for GitHub compatibility
